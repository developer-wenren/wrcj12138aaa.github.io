Java 修炼之道:Maven 常用命令清单

2018-10-18 / 2 min read

前言

Maven 代替 Ant 作为 Java 项目的构建工具已经很久,除了会在 IDE 的图形化界面上操作 Maven, 学会命令行方式操作也十分重要. 这里先记录下基本的Maven命令,后续新增再来更新.

Mavn 常用命令
清除 mvn clean
打包 mvn package
发布到本地 mvn install
发布到线上 mvn deploy
把依赖复制到指定目录 mvn dependency:copy-dependencies -DoutputDirectory=./lib
部署 JAR 包到私服 mvn deploy:deploy-file -DgroupId=X -DartifactId=-X -Dversion=X -Dpackaging=jar -Dfile=X -Durl=X
执行指定类的 main方法 mvn exec:java -Dexec.mainClass=X
查看依赖树 mvn dependency:tree
执行指定的测试用例 mvn test -Dtest=X#Y X 为类名, Y 为方法名,可以用 * 表示所有
不编译测试用例进行操作 mvn -Dmaven.test.skip=true ...
使用指定 POM 文件运行 mvn -f X ... X 为POM 文件路径
日志输出错误信息 mvn -q ...
传递给 POM参数 mvn -DNAME=name 在POM 中使用${NAME}方式获取
