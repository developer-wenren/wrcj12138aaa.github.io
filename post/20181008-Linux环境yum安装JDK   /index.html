Linux环境安装JDK-yum方式

2018-10-08 / 2 min read

前言

为了在公司能搭建一个系统服务系统, 需要在一台新的阿里云服务器上搭建持续集成环境,却发现连基本的Java环境都没有,上网一搜各种五花八门的安装方式眼花缭乱,这次还是通过自己实践记录方式输出到博客,方便以后遇到类似情况需要搭建Java环境时使用.话不多说,马上进入正题.

正文

环境基础

操作系统 CentOS 7.4 64位

1. 下载rpm文件

在Oracle官网上下载对应JDK版本的rpm文件

2. 上传rpm文件

利用 FTP 工具将 rpm文件上传至远程服务器的root目录下

3. 使用 rpm 命令安装 JDK
rpm -ivh jdk-8u181-linux-x64.rpm
4. 设置JDK环境变量
4.1 编辑profile文件
vi /etc/profile
4.2 在打开的profile文件添加如下内容:
JAVA_HOME=/usr/java/jdk1.8.0_181-amd6
JRE_HOME=/usr/java/jdk1.8.0_181-amd6/jre
PATH=$PATH:$JAVA_HOME:$JRE_HOME
export PATH
4.3 让修改生效
source /etc/profile
5. 验证JDK版本信息

使用java命令获取对应jdk的版本信息

java -version
