Scala入门之项目创建

2018-10-29

前言

本文快速通过 IDEA 创建一个基于 Scala 语言的 HelloWord 程序,以及对 Scala 语言进行基础的介绍.

正文

Scala是什么

[/ˈskɑːlɑː/]

Scala 的设计始于2001年, 由马丁•奥德斯基(Martin Odersky)在瑞士洛桑的洛桑联邦理工学院(EPFL)开始设计的编程语言, 并在2004年的Java社区公布了Scala , 而2016年Scala发布了2.0版本,到目前为止版本已经迭代到了 2.12.5.

而Scala语言又究竟是什么的语言呢 ???

Scala combines object-oriented and functional programming in one concise, high-level language. Scala's static types help avoid bugs in complex applications, and its JVM and JavaScript runtimes let you build high-performance systems with easy access to huge ecosystems of libraries.

上面一段来自于Scala官网的描述,从中就可以对Scala的语言特性得知一二:

  • **支持面向对象和函数式.**面向对象方面,Scala中的每一个值都是一个对象;并且通过子类和组合的方式来实现多重继承;而Scala也是一种函数式语言,每个函数都是一个值,提供轻便的匿名函数语法,支持高阶函数,函数嵌套,模式匹配等函数式语言特性.
  • 静态类型语言. 更安全的静态编译检查;天然支持类型推断,减少显示的类型声明.
  • 能在JVM虚拟机和JavaScript运行环境. 由Scala编写的代码可以被编译生成Java字节码,从而直接允许在Java虚拟机之上,并且与此同时提供了与Java代码交互操作的能力(调用Java库方法/使用Java对象);针对JavaScript运行环境,Scala编译器提供Scala.js框架使得Scala代码编译成JavaScript代码,从而运行在浏览器环境中.

当然,Scala语言本身还有更大的功能特性,比如可扩展性,互操作性,丰富的数据类型, 并发性等等.

为什么要使用Scala

  • 相比Java, Scala的语法灵活,简洁
  • 经过Scala编译器优化增强,性能更高
  • 丰富的语言特性,可扩展性高

能做作为数据挖掘的工程语言,效率非常高; 并且Scala 的 Java 兼容性使它非常适合 Android 的开发, 适合功能性方法;除了能写后台代码,还能编写可执行的前台代码,运行于浏览器中.

环境准备

  • IntelliJ IDEA
  • JDK 1.8
  • Scala SDK 2.12.5

Scala版的HelloWord之旅

1.在IDEA的远程插件库搜索 IntelliJ插件-Scala,选择后安装重启IDEA

2.打开IDEA,新建工程,选择 Scala => IDEA

3.选择好JDK和Scala SDK,命名Scala项目为helloWord,确认创建.

Tip: 如果没有对应的Scala SDK,可以通过Scala SDK 栏右边的 Create... => Download让IDEA去下载最新的Scala SDK,也可以通过手动下载,然后通过Browse...选择本地下载好的Scala SDK 进行关联

3.新建工程后,在src目录下新建Package,包名为demo,在该包名下创建一个Scala文件,为HelloWord.

4.将对应HelloWord.scala文件的代码修改为如下

object HelloWord extends App {
  println("Hello, World!")
}

5.右键HelloWord.scala文件的左边按钮 Run HelloWord ,对这个类文件进行编译运行,得到输出内容: Hello, World!

这样Scala版Hello Word就完成,就是这么简单哈.

要点解析

object 关键字

用来定义单个实例对象的关键字,获取的对象都是单例对象

object IdFactory {
  private var counter = 0
  def create(): Int = {
    counter += 1
    counter
  }
}

object Main extends App {
  var newId: Int = IdFactory.create()
  println(newId) // 1
  var newerId = IdFactory.create()
  println(newerId) // 2
}

extends 关键字

Scala中用来继承其他类的方法和字段的关键字,跟Java的extends一样

Scala系统类 App

trait App extends scala.AnyRef with scala.DelayedInit {
  @scala.deprecatedOverriding("executionStart should not be overridden", "2.11.0")
  val executionStart : scala.Long = { /* compiled code */ }
  @scala.deprecatedOverriding("args should not be overridden", "2.11.0")
  protected def args : scala.Array[_root_.scala.Predef.String] = { /* compiled code */ }
  @scala.deprecated("the delayedInit mechanism will disappear", "2.11.0")
  override def delayedInit(body : => scala.Unit) : scala.Unit = { /* compiled code */ }
  @scala.deprecatedOverriding("main should not be overridden", "2.11.0")
  def main(args : scala.Array[_root_.scala.Predef.String]) : scala.Unit = { /* compiled code */ }
}

Traits 关键字

用来声明Scala特有的一种类型, 该类型能包含一些方法和字段,并且允许进行多个进行结合,并且允许被其他Trait类型或者类继承其方法和字段.

trait Greeter {
  def greet(name: String): Unit = println("Hello, " + name + " !")
}

class CustomizableGreeter(prefix: String, postfix: String) extends Greeter {
  override def greet(name: String): Unit = {
    println(prefix + name + postfix)
  }
}

object HelloWord extends App {
  var greeter = new CustomizableGreeter("[","]");
  greeter.greet("1,2,3,4,5,6,7,9");
  // [1,2,3,4,5,6,7,9]
}

def关键字

用于声明函数的关键字

def greet(name: String): Unit = println("Hello, " + name + " !")

Unit 用于表示函数的返回类型为无意义的值,类似Java中的void,但不同的是Unit也表示着一个值

override关键字

方法重写的关键字,用法与Java的相同
